## 📦 Installation

> 🌐 **Ausführliche Schritt-für-Schritt-Anleitung auf unserer Webseite:**
> **[cali72mero.github.io/ShieldBlock](https://cali72mero.github.io/ShieldBlock/)**

### Chrome / Brave / Vivaldi
1. Repository herunterladen: `Code` → `Download ZIP`
2. `chrome://extensions` öffnen → **Entwicklermodus** aktivieren
3. **"Entpackte Erweiterung laden"** → Ordner `extension/chromium/` wählen
4. **Fertig!** 🎉

### Firefox
1. Repository herunterladen
2. `about:debugging#/runtime/this-firefox` öffnen
3. **"Temporäres Add-on laden"** → `extension/firefox/manifest.json` wählen
4. **Fertig!** 🎉

## 📦 Installation

> 🌐 **Full step-by-step guide on our website:**
> **[cali72mero.github.io/ShieldBlock](https://cali72mero.github.io/ShieldBlock/)**

### Chrome / Brave / Vivaldi
1. Download repository: `Code` → `Download ZIP`
2. Open `chrome://extensions` → Enable **Developer mode**
3. Click **"Load unpacked"** → Select `extension/chromium/` folder
4. **Done!** 🎉

### Firefox
1. Download repository
2. Open `about:debugging#/runtime/this-firefox`
3. Click **"Load Temporary Add-on"** → Select `extension/firefox/manifest.json`
4. **Done!** 🎉
