🌐 Anleitung: Chrome / Brave / Vivaldi 🌐 Guide: Chrome / Brave / Vivaldi

1
Repository herunterladen
Gehe zu github.com/cali72mero/ShieldBlock und klicke auf Code → Download ZIP. Entpacke die Datei. Download repository
Go to github.com/cali72mero/ShieldBlock and click Code → Download ZIP. Extract the file.
2
Erweiterungsseite öffnen
Tippe in die Adressleiste: chrome://extensions Open extensions page
Type in the address bar: chrome://extensions
3
Entwicklermodus aktivieren
Schalte den Entwicklermodus-Schalter oben rechts ein. Enable Developer Mode
Toggle the Developer mode switch in the top right.
4
Erweiterung laden
Klicke auf "Entpackte Erweiterung laden" und wähle den Ordner extension/chromium/ Load extension
Click "Load unpacked" and select the extension/chromium/ folder.
✅
Fertig! 🎉
Das ShieldBlock-Icon erscheint in deiner Toolbar. Klicke darauf, um die Erweiterung zu steuern. Done! 🎉
The ShieldBlock icon appears in your toolbar. Click it to control the extension.
