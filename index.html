🧰 Was ShieldBlock kann

📺

YouTube Werbefrei

Blockiert Pre-Roll, Mid-Roll und Overlay-Ads auf YouTube. Überspringt unskippable Ads automatisch. Keine Werbe-Unterbrechungen mehr.

🐦

Twitter/X Clean

Entfernt Promoted Tweets, Werbe-Trends und Tracking-Pixel. Dein Feed zeigt nur noch echten Content.

🤖

Reddit Ohne Werbung

Blockiert Promoted Posts, Sidebar-Ads und nervige "App installieren"-Popups auf Reddit.

🔍

90+ Tracker blockiert

Google Analytics, Facebook Pixel, Hotjar, Mixpanel, Segment und über 80 weitere Tracking-Dienste werden komplett blockiert.

🍪

Cookie-Banner weg

Entfernt Consent-Banner von Cookiebot, OneTrust, Didomi und allen großen Anbietern automatisch.

🚀

Blitzschnell

Nutzt die moderne declarativeNetRequest API (wie uBlock Lite). Blockiert auf Netzwerkebene, bevor Werbung überhaupt geladen wird.

🌐 Installation: Chrome / Brave / Vivaldi

1. Lade das Repository herunter oder klone es mit git clone
2. Öffne chrome://extensions in deinem Browser
3. Aktiviere den Entwicklermodus (rechts oben)
4. Klicke "Entpackte Erweiterung laden"
5. Wähle den Ordner extension/chromium/
6. Fertig! 🎉 Das Shield-Icon erscheint in deiner Toolbar.

🦊 Installation: Firefox

1. Lade das Repository herunter oder klone es
2. Öffne about:debugging#/runtime/this-firefox
3. Klicke "Temporäres Add-on laden"
4. Wähle die Datei extension/firefox/manifest.json
5. Fertig! 🎉
